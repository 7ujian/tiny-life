任务：
基于参考图中已提取的女孩角色，生成一套“浇水 / 洒水”循环动画精灵图。

要求：
这是角色动画扩展任务，不是角色重设计。

必须严格保持角色与参考图完全一致，包括：
- 相同的脸型与五官
- 相同的发型与麻花辫
- 相同的草帽、绿色帽带、白色小花装饰
- 相同的绿色背带裙
- 相同的米白色内搭
- 相同的棕色靴子
- 相同的绿色洒水壶（带白色花朵图案）
- 相同的头身比
- 相同的轮廓
- 相同的描边粗细
- 相同的平涂方式
- 相同的低饱和奶油色系配色

风格要求：
休闲手游农场游戏风格，
柔和奶油色，
暖米色，
低饱和橄榄绿，
平光照明，
极少纹理，
边缘清晰，
简洁二维游戏美术风格。

禁止：
- 不要重设计角色
- 不要改变服装
- 不要改变视角
- 不要改变透视
- 不要改变比例
- 不要新增配饰
- 不要改成绘本风
- 不要改成概念图风
- 不要改成吉卜力风
- 不要做厚涂
- 不要做写实化

动画要求：
生成一套平滑循环的浇水动画。
动作轻柔、可爱、适合手游。
包含：
- 轻微身体起伏
- 手臂小幅运动
- 水壶前倾
- 出现水流
- 再回到起始姿势形成循环

输出要求：
输出16帧动画精灵图集，
纯白背景，
无场景背景，
无UI，
无文字，
所有帧整齐排列，
角色尺度一致，
对齐一致，
适合直接用于2D游戏动画。
